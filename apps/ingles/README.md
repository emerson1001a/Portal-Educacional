# Modulo de Ingles

Prototipo inicial do modulo de Ingles do Portal Educacional.

## Escopo

- Sem IA obrigatoria.
- Exercicios gerados por templates controlados.
- Missao com blocos.
- Cada bloco tem um unico formato.
- Primeiro teste com `verb to be`, `simple present` e `present continuous`.
- Nivel de apoio configuravel pelo adulto.
- Feedback curto de erro com regra e resposta modelo.

## Rodar localmente

```bash
npm run static
```

Abrir:

```text
http://localhost:4188
```
